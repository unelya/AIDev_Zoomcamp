## Homeworks

### 01-ToDoDjango
A simple To-Do application built with the Django framework and a "Yulia-gull" background.  
It supports:
- Creating TODO items
- Editing TODO items
- Deleting TODO items
- Setting due dates
- Marking TODOs as completed
- A basic UI layout and CSS styling
- Full automated tests

### 02-OnlineCodingInterviews
A platform for online coding interviews.  
It supports:
- Creating a link and sharing it with candidates
- Allowing all connected users to edit code collaboratively
- Showing real-time code updates to all participants
- Syntax highlighting for multiple programming languages
- Safe code execution directly in the browser
- Live deployment via Render

Deployed at: https://onlinecodinginterviews.onrender.com

### 03-MCP-Context
A FastMCP server that combines the Jina reader with zip-backed document search.  
It supports:
- Fetching any web page as Markdown via Jina reader (`fetch_page_markdown`)
- Searching `.md` and `.mdx` documents inside local ZIP archives (`search_docs`)
- Automatic path cleanup (removes top-level folder in ZIP entries)
- Reusing a cached index for fast repeated searches
- STDIO client support for quick terminal usage
